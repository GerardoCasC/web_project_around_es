# Tripleten web_project_around_es

Proyecto de sprint 8: Alrededor de los EE.UU., parte 3 — Clases de TypeScript y refactorización POO

Este proyecto cuenta con una función para renderizar diferentes tipos de cartas dentro de un perfil. También tenemos la opción de agregar mas cartas dinámicas, con interacciones como elimintar y dar like. Aparte se puede editar la información del perfil del usuario. Ambos casos con formularios dinamicos mediante Popups con validación personalizada.

Dentro de las tecnologías utilizadas están: HTML, CSS, TypeScript, y la estructura esta enfocada en POO
